# AI-Based Crop Disease Detection and Farmer Recommendation System

## Project Overview

This project is an AI-powered crop disease detection system that identifies plant leaf diseases from images using Deep Learning. The system predicts the disease, displays confidence scores, shows Top-3 predictions, and provides farmer-friendly recommendations such as symptoms, treatment, prevention, pesticide/fungicide, and fertilizer suggestions.

The project is built as an end-to-end Machine Learning application using TensorFlow, Transfer Learning, and Streamlit.

---

## Features

- Plant disease detection from leaf images
- Image upload and camera capture support
- CNN and Transfer Learning using MobileNetV2
- Fine-tuned model with improved validation accuracy
- Top-3 prediction analysis
- Confidence score visualization
- Farmer recommendation system
- Pesticide/Fungicide and fertilizer suggestions
- Confusion matrix and classification report
- Interactive Streamlit web application

## Dataset

Dataset used: PlantVillage Dataset

The dataset contains healthy and diseased plant leaf images for crops such as tomato, potato, and bell pepper.

Dataset link:  
https://www.kaggle.com/datasets/emmarex/plantdisease

---

## Model Performance

The model was improved using Transfer Learning and Fine-tuning with MobileNetV2.

Final validation accuracy: approximately 88–89%

Evaluation techniques used:

- Accuracy and Loss Graphs
- Confusion Matrix
- Classification Report
- Top-3 Prediction Analysis
